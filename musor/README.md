# Musor.ku - React + Tailwind Website

Современный веб-сайт для компании по вывозу мусора, переделанный с Tilda на React + Tailwind CSS.

## 🚀 Технологии

- **React 18** - UI библиотека
- **Tailwind CSS** - CSS фреймворк
- **React Router** - маршрутизация
- **PostCSS** - обработка CSS

## 📦 Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект в режиме разработки:
```bash
npm start
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере


## 🔧 Настройка

### Tailwind CSS
Конфигурация находится в `tailwind.config.js`. Основные настройки:
- Кастомные цвета для брендинга
- Адаптивные breakpoints
- Кастомные утилиты

### PostCSS
Настроен в `postcss.config.js` для обработки Tailwind CSS.
